    <h1>Política de Privacidad</h1>
    <p class="secLead">Última actualización: 2026-02-18</p>

    <div class="box">
      <p><b>Resumen:</b> ColmenasPro está diseñada para funcionar <b>en local</b>. No envía tus datos a servidores externos de forma automática. El <b>backup</b> lo controlas tú.</p>
    </div>

    <h2>1. Qué datos maneja la app</h2>
    <p>ColmenasPro permite registrar información apícola introducida por el usuario, por ejemplo:</p>
    <ul>
      <li>Colmenas y núcleos</li>
      <li>Visitas y observaciones</li>
      <li>Producción</li>
      <li>Genealogía/linajes</li>
    </ul>

    <h2>2. Dónde se guardan los datos</h2>
    <p>Los datos se almacenan <b>en el dispositivo</b>. La aplicación está pensada para poder usarse sin conexión.</p>

    <h2>3. Copias de seguridad</h2>
    <p>La app puede ofrecer funciones de <b>exportación/restauración</b> para copias de seguridad. Estas copias las gestiona el usuario (por ejemplo, guardándolas en su almacenamiento, PC o nube personal).</p>
    <p>ColmenasPro no realiza copias automáticas en servidores externos (salvo que en el futuro se active una función explícita y se actualice esta política).</p>

    <h2>4. Beta privada</h2>
    <p>Si solicitas acceso a la beta y nos facilitas un correo electrónico, se utilizará <b>únicamente</b> para gestionar la beta (enviarte el acceso y comunicaciones relacionadas). No se vende ni se comparte con terceros.</p>

    <h2>5. Analítica y terceros</h2>
    <p>En la configuración actual, ColmenasPro no incluye seguimiento comercial del usuario. Si en el futuro se añadiera analítica, se informaría aquí de forma transparente.</p>
